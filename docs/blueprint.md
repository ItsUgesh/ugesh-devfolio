# **App Name**: Ugesh Devfolio

## Core Features:

- Project Showcase: Display detailed project information, including title, description, tech stack, and live demo links, with modern card designs and hover effects.
- Interactive Skillset Display: Present categorized technical skills for Backend, Frontend, CMS, and Tools using clear visual indicators.
- Dynamic Theming: Allow users to effortlessly switch between a dark and light mode for optimal viewing preference.
- Animated Hero Section: Engage visitors immediately with an eye-catching landing section featuring smooth animations, developer's details, tagline, and call-to-action buttons.
- Comprehensive 'About Me' & Services: Provide a brief introduction, experience summary highlighting Laravel, PHP, and WordPress, and a list of offered services.
- Integrated Contact & Social Links: Offer a clean contact form with validation and direct links to email, GitHub, and LinkedIn profiles for easy communication.
- SEO-Optimized Content Tool: An AI tool to suggest and refine project descriptions and taglines for enhanced clarity and search engine visibility.

## Style Guidelines:

- A dark primary scheme to convey professionalism and a premium feel. Primary text and UI elements will be a modern, deep blue (#5199E0) for a tech-inspired look. The background will be an extremely desaturated, dark blue-gray (#161A1D) to provide depth. A vibrant cyan (#75FFFF) will serve as an accent color for interactive elements and highlights, creating strong contrast.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, techy, and bold statement. Body font: 'Inter' (sans-serif) for neutral, highly legible text. For code snippets, 'Source Code Pro' (monospace sans-serif) will be used to ensure readability for technical content.
- Utilize sleek, minimal, line-based icons throughout the design for skills display, social links, and other UI elements to maintain a professional and clean aesthetic.
- Employ a clean, modern, and mobile-first responsive layout with ample whitespace. Content sections will be clearly delineated, and project showcases will use a grid-based card design. A sticky navigation bar will ensure easy access to all sections.
- Incorporate subtle, purposeful motion effects including smooth scrolling, scroll-triggered content reveals, and interactive element transitions. The hero section will feature eye-catching, smooth animations to create a dynamic first impression.
